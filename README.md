Fully responsive design
Animated hero section
Coffee menu cards
Membership plans section
Gallery section

Modern cafe landing page

HTML5
CSS3
JavaScript

This project was developed as a modern local business website to showcase frontend web development skills, responsive design, UI/UX concepts, and interactive website creation.
